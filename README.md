# RECIPE API STORE AND RETRIEVE COOKING RECIPES
A **Recipe API project** allows users to search, view, and access different recipes along with ingredients, cooking steps, and nutritional details.
It fetches recipe data from an external API and displays it in an easy-to-use interface for browsing and meal planning.
softeware requirements:
Programming language:javascript(node.js) 
Framework:express.js
Database:MongoDB
API testing tool:Postman
os:windows
